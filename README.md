## ğŸ“¦ Installation
### Prerequisites
- Node.js v20+
- PostgreSQL database
- Yarn package manager

### Setup Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/sewing-division.git
   cd sewing-division
   ```

2. Install dependencies:
   ```bash
   yarn install
   ```

3. Configure environment variables:
   ```bash
   cp .env.example .env
   ```

   ```bash
   yarn dev
   ```
